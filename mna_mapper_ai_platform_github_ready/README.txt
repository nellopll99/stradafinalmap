# M&A Mapper AI – Streamlit Tool

## ✅ Requisiti
- Python 3.8+
- Chiave API OpenAI valida (gpt-3.5-turbo o gpt-4)
- openai==0.28.1 (compatibile con la sintassi di questo progetto)

## 🚀 Installazione

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 📦 Prompt consigliati
Esempi:
- CRM per PMI nel settore legale in Germania
- Piattaforme SaaS HR in Italia
- Software AI per agricoltura in Europa

## ⚠️ Nota
Non aggiornare la libreria openai alla versione >=1.0.0 senza modificare il codice.
